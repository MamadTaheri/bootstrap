# Display in bootstrap 5 [Link](https://getbootstrap.com/docs/5.0/utilities/display/)

## How it works
It Change the value of the display property

 ## Notation

 `.d-{value}` for xs

`.d-{breakpoint}-{value}` for sm, md, lg, xl, and xxl.

## values
* none
* inline
* inline-block
* block
* grid
* table
* table-cell
* table-row
* flex
* inline-flex

## Display in print

* .d-print-none
* .d-print-inline
* .d-print-inline-block
* .d-print-block
* .d-print-grid
* .d-print-table
* .d-print-table-row
* .d-print-table-cell
* .d-print-flex
* .d-print-inline-flex
